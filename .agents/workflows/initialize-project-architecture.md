---
description: Initializes the base Next.js project with Tailwind, Framer Motion, and the Kommunarka dark industrial design system.
---

1. Initialize a new Next.js project using App Router.
2. Install Tailwind CSS and Framer Motion.
3. Configure Tailwind's `tailwind.config.js` to include the Kommunarka color palette: Background (#111111), Primary Text (#F4F4F4), Accent (#CC4A22), Secondary Text (#6B6B6B).
4. Set up a global layout with a smooth-scroll wrapper (e.g., using Lenis or a similar approach).
5. Create a basic minimal Navigation bar with links: "Archive", "Exhibitions", "About", "Contacts".
6. Apply extreme whitespace and avoid rounded corners (`rounded-none`).
